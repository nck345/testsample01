# REANTY - DỰ ÁN BẤT ĐỘNG SẢN CAO CẤP (FASTCODING VN CODING TEST)

Dự án này được xây dựng như một bài kiểm tra kỹ năng lập trình Frontend (Coding Test) ứng tuyển vào vị trí lập trình viên tại **Fastcoding VN**. 

Dự án hiện thực hóa giao diện website giới thiệu bất động sản **Reanty** chuẩn responsive, tối ưu hiển thị trên cả PC và thiết bị di động (Smartphone).

---

## 🚀 Tính Năng Nổi Bật

- **Responsive Toàn Diện**: Giao diện co giãn tự động linh hoạt từ màn hình máy tính lớn (`1920px`) cho tới các dòng điện thoại thông minh phổ biến (`320px` - `414px`).
- **Semantic HTML5**: Sử dụng 100% các thẻ ngữ nghĩa của chuẩn HTML5 (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`) tăng khả năng tối ưu SEO và cấu trúc code sáng sủa.
- **Vanilla CSS & JS**: 
  - Hoàn toàn viết bằng mã CSS3 thuần, sử dụng biến CSS (`:root` variables) quản lý mã màu và typography đồng bộ.
  - Sử dụng mã JavaScript thuần (không thư viện) để xử lý mở rộng thanh Menu Mobile, chuyển đổi bộ lọc sản phẩm (House 1, 2, 3) và gửi form liên hệ.
- **Đường dẫn tương đối (Relative Paths)**: Toàn bộ liên kết tài nguyên sử dụng đường dẫn tương đối, đảm bảo dự án chạy ổn định ngay lập tức ở mọi máy chủ lưu trữ (Vercel, GitHub Pages, Netlify).
